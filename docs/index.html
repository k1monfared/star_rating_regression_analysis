Review funnel explorer

Completion rate and review length by the star rating a user intends to give.

This is built on synthetic data.
These are raw (unadjusted) segment figures for exploration. The adjusted regression estimates that separate the star effect from confounders are in the project README and outputs. Both outcomes hold their shape across every segment: completion is U-shaped (higher at 1 and 5), length is inverted-U (shorter at the extremes).
